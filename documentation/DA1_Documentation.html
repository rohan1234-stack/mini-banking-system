Problem Statement

Create a simple banking console system that supports account creation, deposits, withdrawals, account summary display, search by account number, persistent storage of accounts, and a transaction log containing account number, transaction type, amount, and timestamp/date.

Feature List

  1. Create account with AccNo, Name, and Balance.
  2. Deposit money into an existing account.
  3. Withdraw money while preventing negative balance.
  4. Save account data persistently in browser local storage.
  5. Download account data as accounts.json.
  6. Maintain transaction log and download it as transactions.json.
  7. Display all accounts in a summary table.
  8. Search any account by account number.
  9. Display the last 5 transactions for the selected account.

File Format Description

Account File: accounts.json

Each record stores account number, name, and current balance.

[
  {
    "accNo": "1001",
    "name": "Anika Sharma",
    "balance": 5000
  }
]

Transaction Log File: transactions.json

Each log entry stores account number, transaction type, amount, and timestamp.

[
  {
    "accNo": "1001",
    "type": "deposit",
    "amount": 250,
    "timestamp": "2026-03-29T11:20:00.000Z"
  }
]
